# qrcode
Gerador de QRCode com NodeJS e Express

Para testar o projeto:
---------------------- 

### Clone o repositório
`$ git clone https://github.com/Rodr1go/qrcode.git`

### Entre no diretório do projeto via terminal
`$ cd qrcode`

### Instale as dependências
`$ npm install`

### Inicie a aplicação
`$ npm start`
